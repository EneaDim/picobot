docker/
======

Contiene il materiale Docker per il sandbox runtime locale.

File principali:
- picobot-sandbox.Dockerfile
- picobot-runtime-bootstrap.sh
- picobot-sandbox-entrypoint.sh

Questa cartella definisce il runtime isolato usato per tool come:
- TTS
- STT
- yt-dlp / media processing
- binari locali di supporto
