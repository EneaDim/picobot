# sandbox_web

## Intent
Fetch di URL (HTTP GET) in sandbox con whitelist e limiti.

## IT
- scarica questa pagina: https://example.com
- recupera il testo di https://...
- fai fetch di questo url e dammi il contenuto

## EN
- fetch this url https://example.com
- download page text from https://...

## Explicit tool
- tool sandbox_web {"url":"https://example.com"}

## Negative
- news digest (che fa search+fetch+summarize)
