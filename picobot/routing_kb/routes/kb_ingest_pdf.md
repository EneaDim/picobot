# kb_ingest_pdf

## Intent
L’utente vuole aggiungere un PDF (o document) alla knowledge base locale (ingest / indicizzazione).
È un’azione, non una domanda.

## Italiano
- ingest pdf ./docs/manuale.pdf
- indicizza pdf ./contratto.pdf
- aggiungi questo pdf alla kb: ./docs/report.pdf
- importa documento ./docs/policy.pdf
- /kb ingest ./docs/architettura.pdf

## English
- ingest pdf ./docs/manual.pdf
- index this pdf ./docs/report.pdf
- add document to kb ./docs/policy.pdf

## Notes
Spesso la frase contiene path relativi o assoluti.

## Negative
- domande sul contenuto del pdf (kb_query)
